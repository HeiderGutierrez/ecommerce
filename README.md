# Expression

¡Bienvenido a Expression, tu destino para la moda en línea! En Expression, ofrecemos una amplia gama de ropa de alta calidad y accesorios para satisfacer tus necesidades de estilo. Navega por nuestra tienda y encuentra las últimas tendencias en moda.

## Información del Proyecto

- **Nombre del Proyecto**: Expression
- **Versión**: 0.1.0
- **Privado**: Sí

## Scripts Disponibles

- `npm run dev`: Inicia el servidor de desarrollo de Next.js.
- `npm run build`: Construye la aplicación para producción.
- `npm run start`: Inicia la aplicación en modo de producción.
- `npm run lint`: Ejecuta el linter para mantener el código limpio.

## Instalación y Uso

1. Clona este repositorio en tu máquina local:
git clone https://github.com/HeiderGutierrez/ecommerce.git

2. Instala las dependencias del proyecto:
cd ecommerce
npm install

3. Inicia el servidor de desarrollo:
npm run dev

4. Abre tu navegador y visita [http://localhost:3000/](http://localhost:3000/) para explorar Expression.
